ライセンスキーが追加されました。
今までツールを使っていた方もスクリプトを入れなおしてください。
ライセンスキーを入力せずにいいね爆をするとツールが使えなくなります。



IOS
1. Stay2をインストールする => https://apps.apple.com/jp/app/stay-2-userscript-manager/id1591620171
2. iPhone設定 → Safari → 拡張機能 → Stay2をオンにする
2.5 iPhone設定 → Safari → 拡張機能 → すべてのウェブサイト 許可を選択
3. Stay2を開き、右上の＋マークを押しスクリプトを書くを選択
4. 最初から入力されている文字をすべて削除する
5. このリンクの中身をすべてコピペする => https://carfox1212.github.io/xrrrxs/code2.txt
6. 作成されたGAMETRADE いいねツール v3をアクティブ化する (中止ボタンを押す)
7. いいねしたい商品のページにアクセスする
7.5 ゲームトレードにログインしてない場合はログインする
# 必ず自分のゲームトレードアカウントでログインしてください。ゲートレ垢とライセンスを紐づけします #
8. ライセンスキーが表示されるのでキーを入力しSendボタンを押す
# ライセンス認証テキストが出ない場合は適当に数回違う商品見れば表示されるようになります #
# この動作をしないと転売ツールとみなされてツールが使えなくなります #
9. サイトのURLの末尾に ?do=0 を追加する 
# すべて半角で入力してください!! 「？」が全角になっている人がいっぱいいます # 
10. 書き足したURLにアクセスする
11. いいね爆ボタンが表示されます
 
PC
1. chromeに拡張機能のtampermonkeyを入れる
2. オプションを開く
3. 新規作成
4. 送られてきたスクリプトをうって保存
5. いいねしたい商品ページにアクセス
6. ライセンスキーが表示されるのでキーを入力しSendボタンを押す
# 必ず自分のゲームトレードアカウントでログインしてください。ゲートレ垢とライセンスを紐づけします #
# この動作をしないと転売ツールとみなされてツールが使えなくなります #
7. サイトのURLの末尾に ?do=0 を追加する 
8. アクセスしたらいいね爆ボタンが表示されます
 
ライセンスキー送信は１回だけで構いません
